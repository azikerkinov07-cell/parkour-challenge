# Parkour Challenge

A fun and challenging 2D platformer built with **Pygame**.

![Game Preview](screenshot.png) <!-- You can add a screenshot later -->

## Features

- **5 unique levels** with increasing difficulty
- **12 lives** to complete the challenge
- **Flight ability** — press `W` or `↑` (1.3 seconds with cooldown)
- Smooth physics and responsive controls
- Clean pixel-art style design

## 🎮 Controls

| Key              | Action              |
|------------------|---------------------|
| `A` / `D` or ← → | Move left / right   |
| `SPACE` or `↑`   | Jump                |
| `W` or `↑`       | Activate Flight     |
